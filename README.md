  - lint-staged
- testing

- styling

  - storybook

  -

## How to use

### Run locally

Install and run the development server:

```bash
yarn install
now dev
```

### Deploy

Deploy it to the cloud with [now](https://zeit.co/now) ([download](https://zeit.co/download))

```bash
now
```
